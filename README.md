# Track Together

Created by Aparna, Tabitha, & Tristan.

A website application built during the **Lancashire Hackathon 2026**.

**You can now access the websit here: [tracktogether.streamlit.app](https://tracktogether.streamlit.app) **

This project was developed within 1.5 days to demonstrate a solution to 1 of 2 prompts:
I.  Make a study tool for students.
II. Make something to help connect students.

Our team built a working website that covers BOTH prompts.

---

## Main Features

- Assignment Tracker
- Competitive Leaderboard
- Connect (chat) Feature

---

## Tech Used

- Framework: Streamlit
- Language: Python (& HTML, CSS)
- Database: SQL
- Hosted on GitHub

---

## Outcome

Our project received an **Honourable Mention** at the hackathon.

## Future Improvements

With more development time, we would expand the project by adding:

- Different levels of account access eg. students, teachers, admins, etc.
- Lecturers have access to their module chats.
- Inputting course & year - autofill's module options.
- Syncing assignments names & dates with blackboard.
- Improving the chat system with full usernames & images.
- Chat moderation.
- Better accessibility eg. screen reader, change font, change font size, etc.
- Added security eg. password hashing.
- Additional input validation.
